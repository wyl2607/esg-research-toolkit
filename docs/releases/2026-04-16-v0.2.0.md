# ESG Research Toolkit v0.2.0 Release

**Release Date**: 2026-04-16  
**Previous Version**: v0.1.0  
**Status**: ✅ Production Ready

## Overview

Version 0.2.0 marks the completion of **Multi-Year Trend Analysis** capability with comprehensive UI/UX enhancements. The toolkit now provides three-year ESG data (2022–2024) for core DAX companies with responsive design, enhanced data export, and improved loading experience.

## Key Features

### 🎯 Multi-Year Trend Analysis
- **5 Core DAX Companies** with complete 2022–2024 data coverage
- **Year-over-Year Comparison** visualizations now display meaningful trend lines
- **34 Company-Year Records** total (2024: 23 companies, 2023–2022: 5 companies each)
- **Benchmark Data** validated for multi-year stability (0 violations)

**Covered Companies:**
- SAP SE (Software)
- BASF SE (Chemicals)
- Volkswagen Group (Motor Vehicles)
- Deutsche Telekom AG (Telecommunications)
- RWE AG (Energy)

### 🎨 UI/UX Enhancements

1. **Responsive Audit Trail** (PeerComparisonCard)
   - Mobile-first responsive layout (1 → 2 → 3 columns)
   - Optimized for all device sizes (mobile, tablet, desktop)
   - Enhanced readability with word-break for long identifiers

2. **Data Quality Warnings**
   - Alerts users when trend data is insufficient (<2 data points)
   - Card-based UI with clear explanation
   - Multi-language support (English, German, Chinese)

3. **Enhanced Data Export**
   - CSV exports now include metadata section
   - Improved filenames with date stamps (prevent overwrites)
   - Better data organization: Metadata → Metrics → Historical Trends
   - Full audit trail for exported data

4. **Skeleton Loaders**
   - Animated loading indicators for BenchmarkPage
   - Improves perceived performance during API calls
   - Dark mode support
   - Reusable component across the app

## Migration Guide

### For Existing Users
No action required. The toolkit maintains backward compatibility with v0.1.0 databases. Multi-year data is automatically available when you access company profiles.

### New Features to Try
1. Navigate to any of the 5 DAX companies (Company Profile page)
2. Scroll to "Historical Trend" section
3. Observe now-meaningful trend lines spanning 2022–2024
4. Export company data to see new metadata section

## Known Limitations

- Trend analysis limited to 5 core DAX companies
- Additional company data collection requires manual report sourcing
- Backend pytest tests require Python 3 dependencies (install with `pip install -r requirements.txt`)
