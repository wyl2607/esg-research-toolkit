# ESG Research Toolkit v0.3.0 Release

**Release Date**: 2026-04-19
**Previous Version**: v0.2.3-beta.1 (2026-04-17)
**Status**: Stable

## Overview

v0.3.0 closes Phase 1 (Foundation) of the roadmap and puts the "year-gap →
auto-fetch → analyst review → merged history" loop in users' hands. The
previous release moved the backend model registry and data integrity into
shape; this one delivers the feature surface that actually changes the
recruiter-facing demo:

- **F1 CompanyYearPicker** — every page that takes a `(company, year)`
  selection now shows which years are already imported vs. missing, and
  deep-links the analyst straight into the upload/gap-fill workflow for the
  missing one.
- **F2 Pending Disclosures lane** — a standalone `/disclosures` workspace
  where auto-fetched reports from official sources (company site, SEC EDGAR,
  HKEX, CSRC/CNINFO) queue up for field-level review before being merged
  into the company history.
- **Multi-year trend backfill** — a reproducible script that drives the F2
  lane end-to-end so core DAX companies actually have 3-year trendlines in
  Company Profile and Benchmark views.
- **Region-aware LCOE defaults** — `/lcoe` now flips currency and reference
  price block based on UI language, with EIA US wholesale prices added for
  the USD path.

## Highlights

### F1 — Gap-aware Company+Year Picker

- New `frontend/src/components/CompanyYearPicker.tsx` (two-step Select:
  company → year) highlights imported years with a `Check` glyph and marks
  missing ones with `Upload`.
- Picking a missing year navigates to `/upload?company=X&year=Y`; Upload
  renders an info banner so the analyst knows why they landed there.
- Backend `GET /report/companies/v2` returns
  `{ company_name, sector, imported_years[], suggested_years[] }`.
- Migrated pages: Frameworks, Taxonomy, Benchmark, Regional, Upload.

### F2 — Pending Disclosures Lane

- `POST /disclosures/fetch` accepts `company_name`, `report_year`,
  `source_type` (`pdf` | `html` | `filing`), and a `source_hints[]` list
  (`company_site`, `sec_edgar`, `hkex`, `csrc`).
- Fetches are attempted per-lane with RPS ≤ 0.5/host, a project user-agent,
  and full attempt auditing (`attempted_urls` per lane + success lane
  recorded on the pending record).
- `GET /disclosures/pending` + new `/disclosures` page let analysts approve
  the whole payload or a subset of metrics, or reject with a note.
- `GET /disclosures/lane-stats` ranks source lanes by recent evidence
  quality so analysts pick the lane most likely to succeed.

### Multi-year Trend Backfill

- `scripts/seed_data/backfill_via_disclosures.py` drives the disclosures
  lane for every company that is missing years inside the seed manifest's
  target window.
- `tests/test_backfill_via_disclosures.py` locks its behaviour.
- Current coverage: **6 / 5** core companies at ≥3 years of history, which
  is enough to unblock the Company Profile trend chart + YoY delta cards.

### Region-aware LCOE

- `LcoePage` auto-selects EUR or USD based on `i18n.language`, swaps the
  default electricity price accordingly, and surfaces a USD reference panel
  (EIA wholesale, 2021-2024) when the user is on the English surface.

### Stability / DX

- **B1-B4 UX bugs** fixed (benchmark banner spacing, BASF 2024 500 caused
  by `evidence_summary` JSON deserialization, hard-coded Chinese dimension
  names in framework scorers, Upload losing deep-link context).
- EU Taxonomy / CSRD / CSRC 2023 `DimensionScore.name` moved to stable
  `snake_case` keys, with frontend `frameworks.dim.*` i18n tables in en,
  zh, de.
- `scripts/automation/converge_worktrees.sh` learned
  `--assert-no-lanes` and `--assert-no-lane-artifacts`;
  `scripts/review_push_guard.sh` now invokes it before allowing a push
  when the repo has been through worktree-based parallel development.

## Breaking Changes

None. The rename of `DimensionScore.name` from bilingual Chinese-first
strings to `snake_case` keys is a breaking change *for external consumers
that pattern-match on that field*, but all in-repo consumers (frontend,
fixtures, tests) are migrated in the same release.
